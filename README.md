# Parcel Starter Kit

## Getting Started

Get a copy of [this repository](https://github.com/braden337/parcel)

```shell
npx degit braden337/parcel my-new-project
cd my-new-project
```

## Install Dependencies

```shell
npm install
```

## Development

```shell
npm start
```

## Deployment

You can deploy with your [Netlify](https://netlify.com) account by running

```shell
npm run deploy
```
